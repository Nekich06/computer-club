# computer-club
## Инструкция по сборке
Для сборки и запуска программы присутствует Makefile для GNU Make.
Он поддерживает следующие цели:
- `build`: сборка исполняемого файла программы, например
```
$ make
```
*или*
```
$ make build
```
Если директории `/bin/` не существует, то она создается сама, а уже в нее помещается исполняемый файл.
- `run`: запуск построенной программы / сборка и запуск программы, например
```
$ make run
```
Для передачи дополнительных параметров используется переменная `ARGS`:
```
$ make run ARGS="somefile.txt"
```
- `clean`: удаляет результат сборки в директории `/bin/`, например
```
$ make clean
```
## Использование тестовых примеров
В корневом каталоге также есть папка `/tests/`, в которой лежат файлы с входными данными, демонстрирующие поведение программы при разных сценариях.
